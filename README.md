## Contexto

Neste projeto, analisei o caso de experiência de atendimento da Hurb com o objetivo de investigar padrões de insatisfação do consumidor, gargalos operacionais e impactos da comunicação na experiência do cliente.

A partir da exploração e visualização dos dados, desenvolvi uma análise orientada por Customer Experience (XP), buscando transformar indicadores de atendimento em insights acionáveis e propostas estratégicas de melhoria.

Para essa análise, utilizei o Excel para organização e estruturação inicial dos dados, SQL para consulta e manipulação das informações e Looker Studio para construção do dashboard interativo e visualização analítica.

## Tecnologias utilizadas

- 📊 Excel
- 🗃️ SQL
